# DemoProject

Unreal Engine 5.5 gameplay demo project focused on action combat, Gameplay Ability System features, AI enemies, UI, inventory, and equipment workflows.

## Overview

This project is a UE5 gameplay prototype built for portfolio and interview presentation. It includes character combat, ability-driven attacks, enemy behavior, item interaction, inventory UI, and equipment-related systems.

## Main Features

- Gameplay Ability System based character abilities
- Projectile, melee, lightning, fire bolt, electrocute, and charge rock ability examples
- Attribute and damage calculation flow
- Enemy AI controller, behavior tree service, and attack task code
- Inventory management, item manifests, item fragments, and grid-based inventory UI
- Equipment component and proxy mesh/equip actor support
- UMG-based HUD, overlay widgets, health bars, spell globes, and damage text

## Tech Stack

- Unreal Engine 5.5
- C++
- Gameplay Ability System
- UMG
- AI Module
- Motion Warping

## Repository Contents

The GitHub repository is intended to contain source assets and project configuration:

- `Config/`
- `Content/`
- `Source/`
- `DemoProject.uproject`

Generated folders such as `Binaries/`, `Intermediate/`, `Saved/`, `.vs/`, and `DerivedDataCache/` are intentionally ignored.

## How To Open

1. Install Unreal Engine 5.5.
2. Clone this repository.
3. Open `DemoProject.uproject`.
4. Regenerate project files if needed.
5. Build the project from Unreal Editor or Visual Studio.
